Personal Robot Photographer Pro

摄影界的智能航空母舰

Camerobot 是一个类人型陪伴摄影机器人：能看懂参考图，复刻主体位置、 构图、机位和光线，并控制底盘、升降、相机头、轻摇臂、灯光和无人机。

拍摄完毕 预览窗口

Product Scope

先替代一部分独立摄影师的重复性工作

个人写真 / 旅拍

自动构图、主体跟踪、倒计时、补光和拍后预览，让一个人也能完成稳定拍摄。

Vlog 摄影助理

跟随移动、保持主体比例、显示录制状态，并用眼睛屏提示用户怎么站。

产品和活动拍摄

复用固定机位、灯光、角度和运镜，完成可标准化、可重复的出片流程。

Software Loop

从参考画面到机器人动作

MVP0 先跑通软件闭环：上传/登记参考图，生成结构化分析，再输出底盘、 升降、云台、相机、摇臂、灯光和眼睛屏事件。

  1. 01参考图 / 视频上传
  2. 02主体、构图、机位、光线分析
  3. 03生成 Shot Plan
  4. 04控制身体各部位
  5. 05眼睛屏显示拍摄状态和预览
